# STM32 Robotics Learning

A 10-week embedded systems and robotics curriculum using the STM32 Nucleo-F446RE.
Each week builds on the last, ending in an autonomous line-following robot with PID control.

## Hardware
- STM32 Nucleo-F446RE
- GY-521 (MPU-6050) IMU
- Raspberry Pi Zero W + 32GB SD card
- L298N motor driver
- TT motors 3–6V 2WD
- MG90S servo x2
- NEMA 17 stepper + A4988 driver
- HC-SR04 ultrasonic sensor x2
- IR reflectance sensor array
- LM2596 buck converter
